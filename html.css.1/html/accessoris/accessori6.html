ANBERNIC RG35XXSP

La ANBERNIC RG35XXSP és una consola portàtil retro compacta amb un disseny plegable tipus clamshell, inspirat en la clàssica Game Boy Advance SP. Equipada amb una pantalla IPS de 3,5 polzades amb resolució de 640x480, ofereix una qualitat d’imatge clara i vibrant, ideal per a jocs retro. Disposa d’un processador Cortex-A9 de quatre nuclis, 256 MB de RAM DDR3, i utilitza el sistema operatiu Linux, compatible amb una àmplia gamma d’emuladors com NES, SNES, GBA, PS1, entre d’altres. La seva bateria de 3.300 mAh permet diverses hores de joc, i la seva doble ranura per a targetes microSD facilita la gestió del sistema i de la biblioteca de jocs. És una opció molt atractiva per als qui busquen portabilitat, nostàlgia i una experiència de joc retro de qualitat.
